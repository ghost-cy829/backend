# NaijaRemitLock - Backend API

Node.js backend API for the NaijaRemitLock dApp.

## What it does

This API serves as a bridge between the frontend and the Stellar network. It provides endpoints for the frontend to get lockbox data.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | / | API status |
| GET | /lockboxes | Get all lockboxes |

## Running Locally

npm install

npm start

Server runs on http://localhost:3000

## Tech Stack

- Node.js
- Express
- Stellar SDK (coming soon)
